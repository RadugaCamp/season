# Сезоны (каждый год)

_Служение детям, подросткам, молодёжи._

**season** - время года; закалять, тренировать; акклиматизировать. (Всё это умещается в одном слове. С чего-то же всё началось...)

## Организация выезда в сезоне

- подготовка
- информирование
- проведение
- возвращение
- встреча после сезона
